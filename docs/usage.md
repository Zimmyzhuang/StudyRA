# Usage Guide

A quick-start guide for using Recallify.

## Creating a Subject

1. Open the sidebar (visible by default on desktop)
2. Click **+ New Subject** at the bottom of the subject list
3. Enter a name and pick a color

## Creating a Note

1. Select a subject from the sidebar
2. Click **+ New Note** or use the keyboard shortcut
3. Start typing in the block editor — the note auto-saves as you write

## Using the Block Editor

- **Headings** — Type `#`, `##`, or `###` followed by a space
- **Bullet list** — Type `-` or `*` followed by a space
- **Task list** — Type `[]` followed by a space
- **Code block** — Type `` ``` `` and press Enter
- **Highlight** — Select text and choose highlight from the bubble menu

### Block Handles

Hover over any block to reveal a ghost handle on the left. Use it to:

- **Drag** to reorder blocks
- **Click** to open a menu for converting block types or deleting

## Keyboard Shortcuts

| Shortcut   | Action                    |
| ---------- | ------------------------- |
| `Cmd+K`    | Open command palette      |
| `Cmd+N`    | New note                  |
| `Cmd+S`    | Force save                |
| `Cmd+B`    | Bold                      |
| `Cmd+I`    | Italic                    |
| `Cmd+E`    | Code (inline)             |

## Spotlight Search

Press `Cmd+K` to open the spotlight command palette. Search for notes by title, switch subjects, or trigger actions — all without leaving the keyboard.
