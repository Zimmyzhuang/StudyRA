## [0.1.0] - 2026-02-07

### Added
- Initial project setup with Tauri 2 + React 18 + TypeScript
- Block-based editor powered by TipTap with rich formatting
- "Hover & Handle" UX with ghost handles to convert/reorder blocks
- Bubble Menu contextual toolbar on text selection
- Subject management with color-coded subjects
- Spotlight search via `Cmd+K`
- Auto-save with debounced writes
- Keyboard-first navigation
- Offline-first SQLite storage with localStorage browser fallback
